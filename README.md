Exercício Banco de Dados (EBAC)
Este diretório contém as consultas prontas para o exercício de banco de dados.

Arquivos
consultas-ebac.sql: script SQL para SQLite/DBeaver (criação da tabela, inserção de dados e consultas).
consultas-ebac-mongo.js: script MongoDB para Compass/mongosh (inserção de documentos e consultas).
Como usar
SQL (DBeaver / SQLite)
Abra o DBeaver e conecte em um banco SQLite.
Execute o arquivo consultas-ebac.sql.
Rode os SELECT para visualizar os resultados.
MongoDB (Compass / mongosh)
Conecte no MongoDB.
Execute o arquivo consultas-ebac-mongo.js.
Rode os find para visualizar os resultados.
